SunPlus SoC producer (https://www.sunplus.com/)

## Dependances

https://git.openembedded.org/openembedded-core
    Core

## Description

SoCs supported: sp7021.

Contains:
- a926 Xboot;
- a926 FW sample (B-Chip);
- U-Boot;
- Kernel;
- ISPBOOOT.BIN assemble helper;
- base image files.
